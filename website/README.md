# Ticker Flip Website

Static public website for Ticker Flip.

## Local preview

```sh
python3 -m http.server 4177 --directory website
```

Then open `http://localhost:4177`.

## Deploy

Upload the contents of this `website` folder to a static host, GoDaddy file manager, Netlify, Vercel, Cloudflare Pages, or Render Static Site. The API is separate and lives at `https://api.tickerflip.com`.
