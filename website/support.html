Support

Quick fixes before the panic button.

Most early issues come from Bluetooth pairing, permissions, or cloud sync settings. Start here.

API status

Check whether the backend is awake here: https://api.tickerflip.com/health. If it says connected, the API can reach the database.

Heart-rate strap does not show up

Wear the strap, make sure the sensor is awake, wet the contacts if needed, and keep it close to the phone. Ticker Flip tries to show heart-rate devices, not every random Bluetooth device in the room.

Recording saved but sync failed

If the app says local save worked but API sync failed, the video or session still saved on the phone. Check Wi-Fi or cellular, then try syncing again later.

No video or microphone access

On iPhone, go to Settings, find Ticker Flip, and make sure Camera, Microphone, Bluetooth, and local network permissions are enabled when needed.

Need help?

Email support@tickerflip.com. Include your iPhone model, iOS version, strap model, and what you tapped before the issue happened.
